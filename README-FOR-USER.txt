SCHOOL STUDENT MANAGEMENT APP
============================

Thank you for using the Student Management App! This application helps teachers manage student profiles, attendance, health information, marks, and results.

QUICK START
===========

1. EASY WAY TO START THE APP:
   - Double-click on START-APP.bat
   - Wait for the server to start (you'll see a message with the URL)
   - Open your web browser and go to http://localhost:5175

2. CHECK IF APP IS RUNNING:
   - Double-click on check-app.bat
   - This will tell you if the application is currently running

FEATURES
========

✓ Student Profile Management
  - Store student information including name, gender, caste, date of birth, etc.
  - Class selection (1-10)
  - Permanent storage using your browser's local storage

✓ Attendance Tracking
  - Monthly attendance recording
  - Select months from June to May

✓ Health Information
  - Track height, weight, and BMI
  - Health notes section

✓ Marks Entry
  - Term selection (Term 1/2)
  - Subject-wise marks entry
  - Automatic grade calculation

✓ Teacher Dashboard
  - View all student information
  - Export reports
  - Class-wise subject management

✓ Student Results
  - Search students by class
  - View detailed results
  - No delete option in results (as requested)

INSTALLATION
============

For a one-click installation:

1. Download and install NSIS (Nullsoft Scriptable Install System):
   - Go to http://nsis.sourceforge.net/Download
   - Download the latest version
   - Install it on your computer

2. Create the installer:
   - Right-click on setup.nsi
   - Select "Compile NSIS Script"
   - This will create "Student Management App Setup.exe"

3. Run the setup:
   - Double-click on "Student Management App Setup.exe"
   - Follow the installation wizard
   - The app will be installed with desktop shortcuts

TECHNICAL INFORMATION
=====================

This is a modern web application built with:
- React (with Vite for fast development)
- JavaScript
- HTML5
- CSS3

The application runs locally on your computer and does not require an internet connection after installation.

TROUBLESHOOTING
===============

If the application doesn't start:

1. Make sure Node.js is installed:
   - Download from https://nodejs.org/
   - Install the LTS version

2. If you get errors:
   - Close all command prompt windows
   - Restart your computer
   - Try starting the app again

3. If the app doesn't look right:
   - Try a different web browser (Chrome, Firefox, Edge)
   - Make sure JavaScript is enabled in your browser
